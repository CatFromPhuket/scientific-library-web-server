#  SCIENTIFIC LIBRARY 

REST API для управления научной библиотекой. 

## Описание проекта

Проект веб-сервера для обеспечения доступа к научной литературе. Позволяет хранить информацию об учёных, научных статьях и темах. А также взаимосвязях между ними. Позволяет искать статьи отдельного ученого или на отдельную тему.


## Инструкция по запуску
1.  git clone https://github.com/CatFromPhuket/scientific-library-web-server.git
2.  Установка виртуальной среды 
3.  Установка зависимостей из requirements.txt 
4.  uvicorn main:app --reload (bush для MacOS)
